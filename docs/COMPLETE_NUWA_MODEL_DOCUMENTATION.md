## 🚀 使用方法

### 1. 在Blender中创建模型

```bash

python3 blender_mcp_server.py script complete_nuwa_model.py
```

### 2. 渲染高质量图像

```bash
# 渲染完整女娲模型
python3 blender_mcp_server.py script render_complete_nuwa.py
```

### 3. 在Blender界面中查看

1. 打开Blender
2. 切换到Scripting工作区
3. 加载 `complete_nuwa_model.py`
4. 点击"Run Script"
